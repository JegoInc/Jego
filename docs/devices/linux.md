# Linux 电脑怎么翻墙

Linux严格来说是单指操作系统的内核，因操作系统中包含了许多用户图形界面和其他实用工具。

无忧行的订阅节点服务（大陆用户通常称之为"机场"）与Linux系统上的众多网络代理工具程序（大陆用户通常称之为"梯子"）兼容。只需简单学习，就能轻松让电脑和手机上的所有应用程序都能翻墙，随时畅享全球无限制网络资源。
